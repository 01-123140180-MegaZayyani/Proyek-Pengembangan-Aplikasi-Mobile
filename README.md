# 🍳 Masakuy - Aplikasi Resep Makanan

Masakuy adalah aplikasi resep makanan berbasis Kotlin Multiplatform dengan Jetpack Compose.  
Fitur utama:  

- Menampilkan daftar resep dengan gambar, estimasi biaya, dan waktu masak  
- Filter resep berdasarkan budget  
- Menyimpan resep favorit  
- Menampilkan rekomendasi resep berbasis AI (Ktor + Gemini API)  
- Arsitektur clean + MVVM + Koin DI + SQLDelight + DataStore
